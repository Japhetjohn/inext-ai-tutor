# iNextAI Tutor: The Web3 Learning Companion 🚀

iNextAI is an AI-powered Web3 learning platform that adapts to users' needs, providing interactive learning experiences, quizzes, progress tracking, and AI-driven mentorship.

## *Features*
1. *Dashboard Feature:* Tracks user learning, integrates Web3 wallets, and offers gamification.
2. *Web3 Tutor:* AI-driven structured courses for Web3 beginners, practitioners, and experts.
3. *Personalized Learning Path:* Adapts content based on user knowledge and preferences.

## *Tech Stack*
- *Frontend:* React, Next.js, Tailwind CSS
- *Backend:* Node.js, Python (FastAPI/Django)
- *Blockchain:* Solidity, Web3.js, ethers.js
- *Database:* PostgreSQL, MongoDB, mysql
- *AI Integration:* GPT-based tutoring, AI-generated images
